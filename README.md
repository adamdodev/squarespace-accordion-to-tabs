# Squarespace Accordion to Tabs - Alpha

Convert Accordion blocks to Tabs in Squarespace.

**Note:** This is WIP Alpha code and may need changes before it will work with your exact Squarespace setup.

## Tested Squarespace Versions

This code snippet has been tested working with a number of Squarespace 7.1 websites in April 2026.

Squarespace updates the code for its blocks regularly and this snippet may break at any point. Please ensure you are familiar with CSS and Javascript before using it to ensure you can tweak the code if needed.

## Usage

1. Add an **Accordion** block to your page and populate it with content as usual.
2. Add an **Embed** block to the page (e.g. directly blow the Accordion).
3. Edit the Embed block, choose **Embed As** > **Code Snippet**.
4. Select **Embed Data** and paste the contents of [embed.html](./embed.html) into the code editor.
5. Save the page and test.

## Personalising

The snippet uses Squarespace's `var(--paragraphMediumColor)` for text and borders. All other styles can be easily edited in the snippet itself.
